# How to Run BMAD PM Agent

## Create Product Requirements Document (PRD)

The BMAD PM agent will create a comprehensive PRD from your brainstorming session.

### Steps:

1. **Locate the PM agent file**:
   - Path: `.bmad-core/agents/pm.md`

2. **Read the PM agent instructions**:
   - Contains workflows for creating PRDs, GDDs, and planning

3. **Workflows available**:
   - `plan-project`: Create comprehensive PRD
   - `gdd`: Create Game Design Document (if applicable)
   - `refine-plan`: Iterate on existing plans

4. **Input for PM agent**:
   - Your brainstorming session: `PSYCHEDELIC-BRAINSTORMING-SESSION.md`
   - Strategic analysis: `BMAD-PSYCHEDELIC-ANALYSIS.md`

### What the PM will create:

✅ Executive Summary
✅ Project Scope & Objectives
✅ Technical Requirements
✅ Feature Specifications
✅ Architecture Plan
✅ Timeline & Milestones
✅ Resource Requirements
✅ Success Metrics
✅ Risk Assessment

### Request to PM:

"Using the brainstorming session results in PSYCHEDELIC-BRAINSTORMING-SESSION.md and analysis in BMAD-PSYCHEDELIC-ANALYSIS.md, create a comprehensive Product Requirements Document (PRD) for the Psychedelic Template Stack project."
