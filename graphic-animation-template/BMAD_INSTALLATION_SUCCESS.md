### Keeping BMAD Updated

Since this is a manual installation, to update BMAD:

1. **Backup Your Customizations**
   ```bash
   cp -R bmad/_cfg bmad/_cfg.backup
   ```

2. **Pull Latest BMAD Source**
   ```bash
   cd /Users/seman/Desktop/BMAD-METHOD/BMAD-METHOD-SOURCE
   git pull origin v6-alpha
   npm install
   ```

3. **Re-copy BMAD Files**
   ```bash
   rm -rf /Users/seman/Desktop/graphic-animation-template/bmad
   cp -R bmad /Users/seman/Desktop/graphic-animation-template/
   ```

4. **Restore Your Customizations**
